About Neoserv

Neoserv is a cloud web application that controls NeoPixel LEDs, which are controlled by an Arduino Yun board. As long as the Arduino board is online, anybody can control the LEDs. On the backend, Spacebrew handles the communication between the clients (the Neoserv web app and the Arduino board).

The Arduino program uses a Spacebrew library written specifically for the Yun and its interconnectivity. Additionally, it uses the Adafruit NeoPixel library to translate messages from the Neoserv web application into the bytestream that is sent to the Neopixel LEDs.
